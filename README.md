# Car Rental System

A full-stack car rental platform developed using Core Java, MySQL, HTML, CSS, JavaScript, Bootstrap, and Razorpay API integration. The project allows users to browse vehicles, book rentals, and make secure online payments through an integrated payment gateway.

## Features

- User Authentication & Login System
- Vehicle Listing & Booking Functionality
- Online Payment Integration using Razorpay API
- Responsive User Interface
- Database-driven Application using MySQL
- Admin & User Functionalities
